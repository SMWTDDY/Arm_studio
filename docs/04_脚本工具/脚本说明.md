# 脚本说明

`scripts/` 是仓库唯一脚本根目录。日常训练、数据处理、仿真诊断、Piper 硬件入口都放在这里的功能子目录中；旧的 `agent_infra/Piper_Env/Script/`、`agent_infra/Realman_Env/Script/`、`agent_factory/script/`、`piper_scripts/`、`scripts/legacy/`、根目录兼容壳已经删除。

专项工具按用途分组：

```text
data_tools/     数据转换、数据验证、IK 对比、图像缓存、视频导出
diagnostics/    环境、相机、渲染、位姿跟踪检查
model_tools/    checkpoint 检查
agent_factory/  agent_factory H5 检查、转换和 smoke trial
piper/          Piper CAN、Orbbec、统一采集、单臂/双臂采集与回放
realman/        历史 Realman 环境采集、合并、回放和 smoke test
sim/            ArmStudio 仿真采集
training/       原 Piper_test/script 中仍有价值的训练与验证脚本
```

旧路径下的兼容 wrapper 已删除。请直接使用分组后的真实路径，例如：

```bash
python scripts/data_tools/validate_pose_dataset.py ...
python scripts/diagnostics/viewenv.py
python scripts/model_tools/check_model.py
```

后续新增脚本时，建议遵守：

```text
用户常用入口 -> scripts/
数据处理工具 -> scripts/data_tools/
环境和硬件诊断 -> scripts/diagnostics/
模型检查工具 -> scripts/model_tools/
agent_factory 辅助脚本 -> scripts/agent_factory/
Piper 硬件/采集/回放 -> scripts/piper/
Realman 历史环境脚本 -> scripts/realman/
仿真采集 -> scripts/sim/
训练和评估脚本 -> scripts/training/
```
